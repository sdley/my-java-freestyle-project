## Getting Started

### Prerequisites

Before you begin, ensure you have the following installed on your system:

- **Java Development Kit (JDK)** - Version 21 or higher (java 25 recommended)
  - Download from [Oracle](https://www.oracle.com/java/technologies/downloads/) or [OpenJDK](https://openjdk.org/)
  - Verify installation: `java -version`

- **Text Editor or IDE** (Choose one):
  - [IntelliJ IDEA](https://www.jetbrains.com/idea/) (Recommended)
  - [Eclipse](https://www.eclipse.org/)
  - [Visual Studio Code](https://code.visualstudio.com/) with Java extensions
  - Or any text editor of your choice

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/sdley/my-java-freestyle-project.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd my-java-freestyle-project
   ```

3. **Start learning!**
   - Browse through the code examples
   - Try running the programs
   - Modify and experiment with the code


## Usage

### Running Java Programs

#### Using Command Line

1. **Compile a Java file**
   ```bash
   javac YourProgram.java
   ```

2. **Run the compiled program**
   ```bash
   java YourProgram
   ```

#### Using an IDE

1. Import the project into your IDE
2. Navigate to the Java file you want to run
3. Click the "Run" button or use the keyboard shortcut (usually Shift+F10 in IntelliJ, Ctrl+F11 in Eclipse)

### Example Workflow

```bash
# Compile a simple Hello World program
javac HelloWorld.java

# Run the program
java HelloWorld

# Expected output:
# Hello, World!
```
